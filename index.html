How I Work

  • Structure first.

    구조를 먼저 설계합니다.
    디자인을 그대로 구현하는 것이 아니라, 유지보수를 고려한 마크업 구조부터 고민합니다.

  • Efficient workflow.

    효율적인 작업 방식을 고민합니다.
    반복 프로젝트에서 공통 구조를 정리하고, 작업 속도와 품질을 함께 개선합니다.

  • Maintainable always.

    항상 유지보수를 생각합니다.
    반복 제작과 운영 환경을 고려해 재사용 가능한 구조를 지향합니다.

  • CMS-aware.

    CMS 환경을 전제로 작업합니다.
    관리자 페이지와 데이터 흐름을 이해하고 화면을 구현합니다.

  • Design aligned.

    디자인 의도를 정확히 반영합니다.
    디자인팀과의 협업을 통해 시안의 의도를 이해하고, 구현 과정에서 발생하는 이슈를 함께 조율합니다.

  • Clear communication.

    명확한 커뮤니케이션을 중요하게 생각합니다.
    기획·디자인·개발 간의 작업 범위를 공유하고 구조를 설명하며 협업합니다.
